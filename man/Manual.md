
# ETL-JS Command Line Interface

This guide provides help using the Command Line Interface (CLI) of ETL-JS.

## Getting Started

### In A Micro-Nutshell

```bash
$ npm install --global @lpezet/etl-js-cli
$ mkdir my_etl
$ cd my_etl
$ etl-js init
$ vi etl.yml
$ echo "Create ETL activities."
$ vi config.yml
$ echo "Configure mods."
$ etl-js run etl.yml
```
